# Tic Tac Toe Game (Java)

This is a simple **Tic Tac Toe game developed in Java** using basic programming concepts.
The game runs in the **console/terminal** and allows two players to play turn-by-turn.

## Features

* Two player game
* Runs in terminal/command prompt
* Simple and easy to understand code
* Demonstrates basic Java programming concepts

## Technologies Used

* Java
* Notepad / Text Editor
* Command Prompt / Terminal

## How to Run the Game

### Step 1: Compile the Program

```cmd
javac TicTacToe.java
```

### Step 2: Run the Program

```cmd
java TicTacToe
```

## Game Rules

1. The game is played on a **3×3 grid**.
2. Player 1 uses **X** and Player 2 uses **O**.
3. Players take turns placing their mark in an empty cell.
4. The first player to get **three marks in a row** (horizontal, vertical, or diagonal) wins.
5. If all cells are filled and no player wins, the game ends in a **draw**.
